Наиболее важные теги

Не обязательно знать ВСЕ ТЕГИ, а лишь наиболее употребляемые, которые дадут возможность выполнить 80% работы. Остальные теги могут быть изучены по мере необходимости и решения той или иной задачи.

Итак, наиболее важные и часто употребляемые:

  • HEAD - шапка сайта. Между этими тегами размещаются все теги meta, этакие инструкции для браузера, скрипты, стили, тег head и другие, предназначенные для не для пользователей, а для браузера.
  • META - размещаются в шапке сайта.
  • BODY - тело сайта. Между этими тегами размещается ВСЁ, предназначенное для пользователя: текст, текстовая разметка, картинки, ссылки.
  • Title - заголовок сайта для поисковика, отображается в строке браузера.
  • H - текстовые заголовки сайта.
  • ul - для создания маркерованного списка.
  • ol - для создания нумерованного списка.
  • li - сам непосредственно список.
  • p - абзац. Между этими тегами располагается текст. У тега есть свои атрибуты, например, выравнивания текста
  • div - это контейнер, может содержать абзацы, объекты, может иметь свой стиль, более массивный блок по содержанию. Используется для придания внешнего вида целому блоку.
  • TABLE - тег для содания таблиц. Раньше таблицы использовались для создания сайта. Сейчас их предпочитают использовать для размещения конкретных данных на странице
  • tr - строка в таблице.
  • td - столбец в таблице.
  • a - тег ссылки. Используется с атрибутом href. Например, создадим несколько ссылок в виде нумерованного списка:
    1. Перейти на главную страницу сайта
    2. Подняться вверх страницы.
    3. Подписаться на мой канал Ютуб
  • strong - выделение текста жирным.
  • em или i - выделение курсивом.
  • br - перенос строки.
  • сenter - выравнивание по центру.
  • font - раньше использовался для изменения характеристик текста, например, цвета. Сейчас для этих целей используют стили.
  • frame - делит окно на части.
  • iframe - между этим тегами можно вставить сторонний код для отображения информации с другого сайта. Например, видео из You Tube:
  • marquee - создает бегущую строку на странице. НАПРИМЕР, ЭТО ВЫГЛЯДИТ ВОТ ТАК
  • strike - перечеркнутый текст.
  • blockquote - выделение цитат.
  • hr - горизонтальная линия.
  • img - тег для размещения изображения на сайте. Например:

  • span - этот тег используется для того, чтобы управлять стилем более мелких элементов, в отличии от тега div. Например, я сейчас сделал этот предложение курсивом при помощи тега span, задав ему определенный класc.

Итак, в данном случае с помощью html кода я разместил текст на странице, немного отформатировал его, но сама страница не имеет стиля. Не задан шрифт текста, отступы, цвет, бекграунд страницы и так далее.

Можно, конечно, к каждому тегу прописать свои атрибуты и стили и таким образом, создать уникальный стиль страницы, но для этого используют CSS - таблицу стилей. Преимущество подключения к документу html таблицы стилей в том, что изменения в CSS влияют глобально на весь сайт. Таким образом, html - это гипертекстовая разметка сайта, a CSS - это стиль сайта.
